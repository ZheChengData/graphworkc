# CNode

## CNode类初始化

Parameters:

| Name         | Type   | Description            | Default |
| ------------ | ------ | ---------------------- | ------- |
| ID           | int    | 节点的编号             | None    |
| PositionX    | double | 节点的X坐标            | None    |
| PositionY    | double | 节点的Y坐标            | None    |
| IncomingLink | vector | 进入节点的路段编号集合 | None    |
| OutgoingLink | vector | 离开节点的路段编号集合 | None    |
